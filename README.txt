------------------------------------------------------------------------------------
To compile the util program in terminal use the below command:

 javac -d bin/ src/com/test/program/util/*.java

it will generate the class file in 'bin' Directory.

------------------------------------------------------------------------------------
To compile the Driver program in terminal use the classpath and below is command:

 javac -d bin/ -cp bin/ src/com/test/program/Driver.java


------------------------------------------------------------------------------------
To execute the program run the below command:

 java -cp bin/ com.test.program.Driver
